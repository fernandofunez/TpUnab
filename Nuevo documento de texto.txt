Clase Usuario
- nombre: str
- email: str
- carpetas: 
+ enviar_mensaje()
+ recibir_mensaje()
+ listar_mensajes()

Clase Mensaje
- remitente: str
- destinatario: str
- asunto: str
- cuerpo: str
+ resumen()

Clase Carpeta
- nombre: str
- mensajes: list
+ agregar_mensaje()
+ listar()

Clase servidor correo
- usuarios: 
+ registrar_usuario()
+ enviar()


Encapsulamiento: 
todos los atributos son privados (_atributo) y se accede mediante métodos o propiedades.
Cada clase tiene un rol claro : Usuario gestiona sus carpetas y mensajes, el Mensaje, encapsula la información del correo, la Carpeta,organiza los mensajes, el Servidor correo,conecta usuarios y maneja el envío.
El sistema puede crecer fácilmente (agregar filtros, etiquetas, búsqueda).
y los métodos como listar() puede usarse en múltiples contextos.
